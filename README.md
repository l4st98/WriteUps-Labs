# 📌 WriteUps-Labs

This repository contains my **technical write-ups** for machines and challenges from:

- HackTheBox  
- TryHackMe  

I am building this repository mainly to **learn**, improve my skills, and document my progress in cybersecurity.  
Everything here is part of my personal study journey — nothing professional, nothing advanced, just me trying to get better step by step.

---

## 🧭 Repository Structure

| Platform     | Categories                     | Description                                  |
|--------------|--------------------------------|----------------------------------------------|
| HackTheBox   | Windows / Linux / Challenges   | Write-ups for HTB machines and challenges    |
| TryHackMe    | Easy / Medium / Hard           | Write-ups for THM rooms by difficulty        |

Each machine contains:

- **README.md** → full write-up  
- **imgs/** → screenshots used in the write-up  

---

## 🧠 Write-up Philosophy

My write-ups follow this structure:

1. Full enumeration  
2. Clear explanation of my thought process  
3. Step-by-step exploitation  
4. Privilege escalation with reasoning  
5. Mitigation and defensive notes  
6. Lessons learned  

I am not trying to look like an expert — I am simply documenting what I learn.
